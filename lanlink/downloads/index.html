Android

Download APK universal · works on any phone

Tap the file after downloading to install. Android may ask you to allow installing from this source the first time — that's normal for apps installed outside the Play Store.

Other Android builds
arm64-v8a (smaller · most modern phones) armeabi-v7a (older) x86_64 (emulator)

Windows
Windows 10 & 11 · 64-bit
Download installer .exe setup

Run the installer. Windows SmartScreen may show a "Windows protected your PC" notice because the app isn't code-signed yet — click More info → Run anyway to continue.

🐧
Linux
x86_64 · AppImage
Download AppImage portable · runs on any distro

After downloading, make it executable and run it: chmod +x LanLink-*.AppImage then double-click it (or run it from a terminal). Needs GTK 3, which ships with most desktop distros.

Other Linux builds
.tar.gz (extract & run ./lanlink)
🍎
macOS
Apple Silicon & Intel
Download .dmg disk image

Open the .dmg and drag LanLink to Applications. Since the app isn't notarized yet, the first launch needs right-click → Open (or System Settings → Privacy & Security → Open Anyway).

📱
iOS
iPhone & iPad · advanced
Download .ipa unsigned

The iOS build is unsigned and meant for sideloading via tools like AltStore or Sideloadly, or your own developer signing. It cannot be installed straight from Safari.
