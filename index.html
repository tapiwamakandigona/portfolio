About

I'm a self-taught developer from Kwekwe, Zimbabwe. I started building out of curiosity — first small scripts, then full products. Now I focus on the full picture: backend architecture, database design, responsive UI, and native packaging so apps run on web, Android, Windows, and Linux from a single codebase.

My stack is React + TypeScript + Supabase for most things, Three.js when I need 3D, and Capacitor + Electron for cross-platform reach. I like building in the open.

Outside of code: game design, system design, and trying to make things that people actually use.
